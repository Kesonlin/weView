## Features

前端工程化演示项目。

- 基于 Vue 框架
- Vitest 实现单元测试
- Eslint + Prettier + Husky 语法检查
- 采用 vite 构建
- Vitepress + Vercel 文档网站搭建

## install

```js
  npm i weView
```
